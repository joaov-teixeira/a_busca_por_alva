# A Busca por Alva

A Busca por Alva é um RPG 2D top-down clássico, inspirado em jogos de GBA como Pokémon e Zelda, com foco em exploração, combate em turnos, puzzles ambientais e uma mecânica central de Coragem.

## Conceito

O Herói viaja de um reino pacífico até o Território Negro para resgatar Alva, uma elfa mágica sequestrada pelo Rei Demônio. A narrativa é transmitida principalmente por elementos visuais, mudanças ambientais e pistas mágicas deixadas por Alva.

## Engine

- Godot 4
- Projeto 2D baseado em tiles
